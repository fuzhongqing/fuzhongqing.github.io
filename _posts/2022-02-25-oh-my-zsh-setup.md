```bash

# ~/.zshrc

plugins=(adb autojump git osx)

```

## adb

#### 前置要求
安装adb命令

#### 功能
- 支持 `adb` 命令自动补全。
- 支持设备列表自动补全 `adb -s`。

## autojump

#### 前置要求

安装 [autojump](https://github.com/wting/autojump)

#### 功能

跳转到包含 `foo` 的目录

```
j foo
```
